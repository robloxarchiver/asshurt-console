# Asshurt Console
A simple console for Sirhurt with basic functionality.

# Macros
- `!INJECT` - Injects Sirhurt, MUST BE IN THE SAME FOLDER
- `!EXIT` - Closes Asshurt Console
- `!KILL` - Kills Roblox
